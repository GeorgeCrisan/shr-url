URL Shortener Microservice

Example creation usage in address bar :

https://shr-url.glitch.me/new/https://www.google.com

Example creation usage in form bellow:

https://www.google.com

Example creation output:

{ "original_url":"https://www.google.com"
"short_url":"ttps://shr-url.glitch.me/8170" }

Usage:

https://shr-url.glitch.me/8170

Will redirect to:

https://www.google.com/
